# Using Editor Tools for Specific Tasks


UnigineEditor tools are available via the [Menu Bar](../../editor2/interface/index.md#menu_bar): to open a tool window, choose *Tools* and click the required tool.


![](tools.png)


The following tools are available:


- *[Video Grabber](../../editor2/tools/video_grabber/index.md)* tool is used for capturing high-quality screenshots and frame sequences. > **Notice:** The *Video Grabber* tool is available only in the *Engineering* and *Sim* versions of UNIGINE SDK.
- *[Impostors Creator](../../editor2/tools/impostors_creator/index.md)* tool is used to create impostors that are rendered instead of real objects.
- *[Randomizer](../../editor2/tools/randomizer/index.md)* tool is used for random placing of various objects (cars, houses, trees and so on) making the environment look more natural.
- *[Batch Rename](../../editor2/tools/batch_rename/index.md)* window used for easy renaming of multiple assets in one click.
- *[Bake Lighting](../../editor2/lighting/gi/bake_lighting/index.md)* window is used for all light baking tasks.
- *[Tracker](../../editor2/tools/tracker/index.md)* tool is used for creating sequences animated over time that allows for adding dynamic gameplay elements or making in-game cinematic scenes.
- *[Cleaner](../../editor2/cleaner/index.md)* tool is used to optimize the project by deleting assets that won’t be used in the final build.
- *[Performance Profiler](../../tools/profiling/profiler/index.md)* tool is used to display performance data in a timeline. It allows for detecting bottlenecks of an application and checking assets and code optimization results.
- *[Microprofile](../../tools/profiling/microprofile/index_cpp.md)* is used to inspect performance.
- *[Content Profiler](../../editor2/assets_optimize/content_profiler/index.md)* is used to facilitate the content optimization.
- *[Landscape Exporter](../../editor2/landscape_export/index.md)* is used to convert [Landscape Terrain](../../objects/objects/terrain/landscape_terrain/index.md) into `.mesh` and `.fbx` files for geometry modification, shadow casting, and use in external applications.
- *[Sandworm](../../editor2/sandworm/index.md)* tool is used to create a [Landscape Terrain](../../objects/objects/terrain/landscape_terrain/index.md)/[Global Terrain](../../objects/objects/terrain/terrain_global/index.md) according to provided geodata. The tool features the following:

  - Processing real world elevation and imagery data by using the GDAL.
  - Using GIS data for procedural generation of details and vegetation, roads, communications, buildings, landmarks, and so on.
  - Supporting multiple data layers aligned by geo-coordinates and used for terrain LODs creation.
