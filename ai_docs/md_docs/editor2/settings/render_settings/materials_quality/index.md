# Materials Quality


This section contains settings related to materials quality.


![Materials Quality settings](materials_quality.png)

*Materials Quality Settings*


| Materials Quality | The quality level of the rendered materials. Depending on the specified level, a certain set of features inside graph-based materials connected to the corresponding input of a ***[Material Quality Switch](../../../../content/materials/graph/node_library/misc/material_quality_switch.md)*** node will be applied. One of the following values: - Low - Medium - High (by default) `Console access:render_materials_quality` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_materials_quality)) |
|---|---|
