# Terrain


The section contains settings that control rendering of terrain.


![Terrain rendering settings](terrain.png)

*Terrain rendering settings*


| Triplanar | The value indicating if [triplanar texture mapping](../../../../objects/objects/terrain/terrain_global/details/index.md#triplanar) is enabled for the [Global Terrain](../../../../objects/objects/terrain/terrain_global/index.md). If disabled, planar UV-mapping is used. **disabled** by default. `Console access:render_terrain_global_triplanar` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_terrain_global_triplanar)) |
|---|---|
| Holes | The value indicating if [decal-based holes](../../../../objects/objects/terrain/terrain_global/index.md#decal_holes) are enabled for the [global terrain](../../../../objects/objects/terrain/terrain_global/index.md). **disabled** by default. `Console access:render_terrain_global_holes` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_terrain_global_holes)) |
