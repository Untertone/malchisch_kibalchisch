# Vegetation Animation Parameters


This section contains vegetation animation parameters applied across the whole world.


![](animation_parameters.png)

*Vegetation Animation Parameters*


| Animation Stem | The global scale for movement amplitude of vegetation stems. Range of values: **[0.0f, inf]**. The default value is : **1.0f**. `Console access:render_animation_stem` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_animation_stem)) |
|---|---|
| Animation Leaf | The global scale for rotation angle of vegetation leaves. Range of values: **[0.0f, inf]**. The default value is : **1.0f**. `Console access:render_animation_leaf` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_animation_leaf)) |
| Animation Scale | The global scale for rotation speed of vegetation leaves. Range of values: **[0.0f, inf]**. The default value is : **1.0f**. `Console access:render_animation_scale` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_animation_scale)) |
| Animation Wind | The direction of wind for all vegetation (grass and trees). It is a multiplier for the stem offset (in grass, stem and leaves materials). **(0.0f, 0.0f, 0.0f)** - default value `Console access:render_animation_wind` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_animation_wind)) |
