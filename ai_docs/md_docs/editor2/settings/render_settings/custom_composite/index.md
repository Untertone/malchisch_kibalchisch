# Custom Composite Materials


This section contains settings related to the custom composite materials.


![Custom Composite Materials settings](custom_composite.png)

*Custom Composite Materials Settings*


| Composite Deferred | Custom composite deferred materials, that are used at the [deferred composite](../../../../principles/render/sequence/index.md#deferred_composite) stage of the rendering sequence. Several comma-separated materials (without whitespaces) can be specified. A [debug material](../../../../content/materials/library/debug/debug_materials/index.md) can be put here in order to be added to the final image |
|---|---|
| Composite | Custom composite materials, that are used for [final screen composition](../../../../principles/render/sequence/index.md#composite) instead of the default ones. Several comma-separated materials (without whitespaces) can be specified. |
