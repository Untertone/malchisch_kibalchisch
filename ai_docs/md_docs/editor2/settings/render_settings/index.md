# Render Settings


The *Render* section of the *Settings* window allows adjusting the rendering settings of the world. It has a set of sub-sections given below.


- *[Screen](../../../editor2/settings/render_settings/screen/index.md)* — screen settings such as VR emulation and stereo modes, virtual resolution, border, etc.
- *[Visibility Distances](../../../editor2/settings/render_settings/visibility_distances/index.md)* — settings related to visibility distances for shadows, reflections, and various types of objects.
- *[Textures](../../../editor2/settings/render_settings/textures/index.md)* — texture-related settings, such as quality, resolution, filtering options, etc.
- *[Lights](../../../editor2/settings/render_settings/lights/index.md)* — lights-related settings, such as lens flares effect, batching parameters as well as per-object limits for forward rendering pass.
- *[Transparent](../../../editor2/settings/render_settings/transparent/index.md)* — settings related to rendering of transparent materials.
- *[Shadows](../../../editor2/settings/render_settings/shadows/index.md)* — settings of the scene shadows.
- *[Global Illumination](../../../editor2/settings/render_settings/global_illumination/index.md)* — the settings of the global illumination including SSAO (screen-space ambient occlusion), SSGI (screen-space global illumination) and bent normals.
- *[SSS](../../../editor2/settings/render_settings/sss/index.md)* — settings of the subsurface scattering.
- *[SSR](../../../editor2/settings/render_settings/ssr/index.md)* — settings of the screen-space reflections (SSR).
- *[Decals](../../../editor2/settings/render_settings/decals/index.md)* — decals-related settings.
- *[SSBevel](../../../editor2/settings/render_settings/ssbevel/index.md)* — settings of the screen-space bevel effect (SSBevel).
- *[SSDirt](../../../editor2/settings/render_settings/ssdirt/index.md)* — settings of the screen-space dirt effect (SSDirt).
- *[Terrain](../../../editor2/settings/render_settings/terrain/index.md)* — terrain rendering settings.
- *[Water](../../../editor2/settings/render_settings/water_ssr/index.md)* — water rendering settings.
- *[Clouds](../../../editor2/settings/render_settings/clouds/index.md)* — clouds rendering settings.
- *[Vegetation](../../../editor2/settings/render_settings/vegetation/index.md)* — vegetation animation parameters applied across the whole world.
- *[Environment](../../../editor2/settings/render_settings/environment/index.md)* — settings controlling environment of the scene (scattering, haze, etc.).
- *[Occlusion Culling](../../../editor2/settings/render_settings/occlusion_culling/index.md)* — settings related to occlusion culling.
- *[Camera Effects](../../../editor2/settings/render_settings/camera_effects/index.md)* — settings of the camera (Motion Blur, DOF, etc.).
- *[Color](../../../editor2/settings/render_settings/color/index.md)* — scene color correction settings.
- *[Buffers](../../../editor2/settings/render_settings/buffers/index.md)* — additional buffer settings.
- *[Streaming](../../../editor2/settings/render_settings/streaming/index.md)* — settings related to resource streaming.
- *[Custom Post Materials](../../../editor2/settings/render_settings/custom_post/index.md)* — custom post materials settings.
- *[Debug Materials](../../../editor2/settings/render_settings/debug/index.md)* — debug materials settings.
- *[Custom Composite Materials](../../../editor2/settings/render_settings/custom_composite/index.md)* — custom composite materials settings.


> **Warning:** Make sure that you've disabled all post-effects ([SSAO](../../../editor2/settings/render_settings/global_illumination/indirect_diffuse/ssao/index.md), [SSR](../../../editor2/settings/render_settings/ssr/index.md), [SSS](../../../editor2/settings/render_settings/sss/index.md), Bloom, [DoF](../../../editor2/settings/render_settings/camera_effects/index.md#dof), Bokeh) on all the multi-monitor configurations.
