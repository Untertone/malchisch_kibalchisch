# Shading Quality


This section contains settings related to shading quality.


![Shading Quality settings](shading_quality.png)

*Shading Quality Settings*


| Shading Quality | The quality level of shading. Depending on the specified level, a certain set of shading features inside graph-based materials connected to the corresponding input of a ***[Shading Quality Switch](../../../../content/materials/graph/node_library/misc/shading_quality_switch.md)*** node will be applied. One of the following values: - Low - Medium - High (by default) `Console access:render_shading_quality` ([API control](../../../../api/library/rendering/class.render_cpp.md#render_shading_quality)) |
|---|---|
