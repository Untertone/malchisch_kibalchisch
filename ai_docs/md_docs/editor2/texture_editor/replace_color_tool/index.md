# Replace Color Tool


![](../replace_color.png) **Replace Color** tool replaces all texture colors with the specified one. It should be used for minor content adjustments right in the scene.


There are no specific settings for the tool. All available settings are described in the [Texture Editor](../../../editor2/texture_editor/index.md#settings) article.
