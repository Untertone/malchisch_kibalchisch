# Pencil Tool


![](../pencil.png) **Pencil** tool allows painting hard-edged lines on a texture. It can be used for pixel-pefect editing of textures.


![](pencil.png)


## Settings


All settings are described in the [Texture Editor](../../../editor2/texture_editor/index.md#settings) article, including **[Rectangle Texture](../../../editor2/texture_editor/index.md#rectangle_texture)** and **[Rectangle Texture Opacity](../../../editor2/texture_editor/index.md#rectangle_opacity)**.
