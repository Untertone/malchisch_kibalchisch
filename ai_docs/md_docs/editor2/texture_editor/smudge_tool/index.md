# Smudge Tool


![](../smudge.png) **Smudge** tool smudges the texture.


## Settings


![](smudge.png)


> **Notice:** You cannot smudge the texture using the **[Stamp](../../../editor2/texture_editor/index.md#stamp)** stroke mode.
