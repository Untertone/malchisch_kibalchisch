# Eraser Tool


![](../eraser.png) **Eraser** tool (**E** hotkey) removes changes.


The *Eraser* tool has almost the same [settings](../../../editor2/texture_editor/index.md#settings) as the *Brush* tool, including the **[Stroke Mode](../../../editor2/texture_editor/index.md#stroke_mode)**: the eraser acts like the brush in the same mode.


## Settings


**Opacity** allows controlling whether to erase changes completely or partially:


| ![](opacity_0.png) | ![](opacity_1.png) |
|---|---|
| *Opacity = 1.0* | *Opacity = 0.5* |
