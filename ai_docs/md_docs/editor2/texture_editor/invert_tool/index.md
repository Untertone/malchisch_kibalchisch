# Invert Tool


![](../invert.png) **Invert** tool inverts texture colors.


## Settings


For details, see the [Texture Editor](../../../editor2/texture_editor/index.md#settings) article.


| ![](invert_0.png) | ![](invert_1.png) |
|---|---|
| *Original Texture* | *Inverted Texture* |


By using the *[Active Channels](../../../editor2/texture_editor/index.md#active_channels)* setting, you can control the color channel inversion.
