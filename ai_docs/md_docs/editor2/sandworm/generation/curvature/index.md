# Curvature


![](curvature.png)


This flag indicates if the generated terrain should be curved (*WGS-84* ellipsoid datum). Curving of the terrain is performed using the *[Geodetic Pivot](../../../../objects/geodetics/geodeticpivot/index.md)*.


> **Notice:** Currently supported only for *[Object Terrain Global](../../../../editor2/sandworm/interface/index.md#terrain_type)* with the *[Built-In](../../../../editor2/sandworm/generation/projection/index.md#built_in)* projection enabled.
