# Add-Ons


An add-on is an extension that can be presented as any engine addition: plugin, library, material, asset pack, etc.


Add-ons can be downloaded from Add-On Store at [https://store.unigine.com/](https://store.unigine.com/). Add-On Store contains add-ons created by UNIGINE and add-ons created and uploaded by users. Depending on the add-on usage conditions, add-ons can be downloaded free of charge or upon payment. For more details, see the [Add-On Store documentation](https://store.unigine.com/documentation).


> **Notice:** If the add-on has been upgraded, its newer version should be downloaded and added to the UNIGINE project manually.
