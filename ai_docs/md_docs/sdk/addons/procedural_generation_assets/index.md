# Procedural Generation Assets


The *Procedural Generation Assets* add-on contains a pack of ready-to-use materials and textures to [customize buildings](../../../editor2/sandworm/sources/buildings/index.md) generated using the [Sandworm](../../../editor2/sandworm/index.md) tool to simplify creation of urban environments:


- 6 variations for brick buildings
- 2 variations for concrete buildings

  ![](buildings_001.png)  ![](buildings_002.png)  ![](buildings_003.png)
## Opening the World


To open the world containing assets for the procedural generation of buildings:


- Download the *Procedural Generation Assets* add-on from [Add-On Store](https://store.unigine.com/).
- Add the downloaded add-on (UPACKAGE file) to your project by dragging it into the project `data/` folder in the Asset Browser. In the *Import Package* window that opens, click the *Import Package* button and wait until the add-on contents are imported.
- In UnigineEditor, check the *[Asset Browser](../../../editor2/assets_workflow/index.md#asset_browser)* window. The material and texture files are available in the `data/buildings/` directory.
