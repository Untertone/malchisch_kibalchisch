# VR C# Template


A configurable C# VR template containing a sandbox-style scene with a set of interactive objects and components that can be manipulated using VR controllers.


- Reusable core VR interaction components
- Input support for VR controllers
- Default scene content for interaction and testing purposes


## Features


The template features a default UNIGINE world with interactive elements that can be manipulated using VR controllers. It is designed to demonstrate key features that can be configured to meet the requirements of a wide range of VR applications.


**Key features:**


**Core VR interaction components:**


- Object rotation component
- Object movement component
- Rope interaction
- Sockets and plugs
- Inventory system
- Firearm interaction (pistol)
- Laser pointer
- Movable and grabbable objects


**Interaction and navigation:**


- Object manipulation using VR controllers
- Laser pointer interaction with objects and UI
- Teleportation and free movement within the scene
- Interaction with GUI elements in VR
