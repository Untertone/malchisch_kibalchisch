# Project Templates


This section describes project templates available with UNIGINE SDK and provides instructions on how to use them.


[![](../projects/templates_sm.png)](../projects/templates.png)


When you create a new UNIGINE project, you have the option to use one of the existing templates as a base for your application or game. UNIGINE project templates contain content, code and additional plugins.


To learn how to create a new project from a template, see the [Creating New Project](../../sdk/projects/index_cpp.md#creation) page.


## Template Setup and Maintenance


In some cases, such as when working offline, you may need to download a template manually for use on a device without a network connection. Templates can be managed directly from the SDK Browser - to download a required template, use the three-dot icon on the template card.


![](img/template_manual.png)


The downloaded template is stored in the `%storage_path%/templates` directory. **Copy the SDK and template** to the `%storage_path%/sdks` and `%storage_path%/templates` directories on another device for **offline use** (for example, when upgrading a non-template-based project created with an SDK version earlier than **2.21**).


Use the same three-dot menu to delete a template that is no longer needed.


![](img/template_manual_delete.png)
