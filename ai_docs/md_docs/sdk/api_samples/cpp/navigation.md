## Navigation Mesh

This sample demonstrates how to configure pathfinding between two points with obstacles using *Navigation Mesh*. The *Route radius* parameter controls the navigation width - increasing it creates wider paths that maintain safe distance from obstacles, while decreasing it allows navigation through tighter spaces.
**SDK Path:***<SAMPLES_PROJECT_PATH>/source/navigation/navigation_mesh*
