# Multi-Display Configurations (SpiderVision)

> **Warning:** The functionality described in this article is not available in the Community SDK edition.
> You should upgrade to [**Sim**](https://l.unigine.com/SdhugY462) SDK edition to use it.


## 2X1 Horizontal Monitor Setup

This sample demonstrates a two-monitor configuration with the possibility to configure distance and angle between the neighboring monitors via the application UI.


The sample shows how to use the API to control the parameters via code. The same parameters are accessible via the **[SpiderVision](../../../principles/render/output/multi_monitor/spidervision_plugin/index.md)** setup window (**F10**).


**SDK Path:***<SAMPLES_PROJECT_PATH>/source/multi_display_configurations_spider_vision/2x1_horizontal_monitor_setup*

The sample shows how to use the API to control the parameters via code.

## 3X3 Monitor Wall Setup

This sample shows how to create a Wall configuration and access the settings that affect image rendering to make the lines between monitors invisible. Toggling the available settings on and off gives a clearer understanding of how various rendering settings affect the overall image.


The sample shows how to use the API to control the parameters via code. The same parameters are accessible via the **[SpiderVision](../../../principles/render/output/multi_monitor/spidervision_plugin/index.md)** setup window (**F10**).


**SDK Path:***<SAMPLES_PROJECT_PATH>/source/multi_display_configurations_spider_vision/3x3_monitor_wall_setup*
## Attach Camera View To Window

## Projection Calibration Overlays

This sample demonstrates image fine-tuning tools such as warping, blending, color correction, and masking.


The sample shows how to control the parameters via API. The same parameters are also available in the **Projection Setup** section of the **[SpiderVision](../../../principles/render/output/multi_monitor/spidervision_plugin/index.md)** setup window (**F10 -> Tools -> Projection Setup**).
