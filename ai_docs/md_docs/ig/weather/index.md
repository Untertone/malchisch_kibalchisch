# Weather Plugin

> **Warning:** The functionality described in this article is not available in the Community SDK edition.
> You should upgrade to [**Sim**](https://l.unigine.com/SdhugY462) SDK edition to use it.


The plugin implements visualization of various weather and environment effects based on the provided data, such as date, time, time zone, cloud type, and other weather conditions.


The [IG Editor Plugin](../../ig/ig_plugin.md) allows you to adjust the weather properties right in the Editor.
