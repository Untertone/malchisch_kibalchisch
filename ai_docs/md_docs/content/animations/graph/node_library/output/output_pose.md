# Output Pose


![](../img/output_pose.png)

### Description

The final output of the animation graph. The pose connected to its input is applied to the character's skeleton. This node is created automatically and cannot be added or removed manually - every animation graph contains exactly one **Output Pose**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/anim_pose.png) | **Pose** | The final pose to apply to the skeleton. |
