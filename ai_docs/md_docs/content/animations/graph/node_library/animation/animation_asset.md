# Animation Asset


![](../img/animation_asset.png)

### Description

References an [animation file](../../../../../content/animations/animation_assets/index.md#animation) (`.anim`) for use in the Animation Graph. Select the reference asset in the **Animation** property, then connect the output to an **[Animation Player](../../../../../content/animations/graph/node_library/animation/animation_player.md)** or **[Animation Pose](../../../../../content/animations/graph/node_library/animation/animation_pose.md)** node to play or sample it.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/anim_asset.png) | **Anim Asset** | The referenced animation asset to be used by playback nodes. |


## Properties


| Animation | The animation file (`.anim, .fbx`) to reference. Selected via the asset browser. |
|---|---|
