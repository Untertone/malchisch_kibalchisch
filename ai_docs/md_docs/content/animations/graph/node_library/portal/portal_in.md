# Portal In


![](../img/portal_in.png)

### Description

Marks an entry point for a hidden connection. Use portals to reduce visual clutter in complex graphs by replacing long connection lines with named portal pairs. Connect a wire to a **Portal In** node, then place **[Portal Out](../../../../../content/animations/graph/node_library/portal/portal_out.md)** nodes wherever you need to access that data.


Multiple **[Portal Out](../../../../../content/animations/graph/node_library/portal/portal_out.md)** nodes can reference the same **Portal In**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The value to pass through the portal. The type adapts automatically to match the connected wire. |


## Properties


| Name | Identifier for the portal (displayed on the node). |
|---|---|
| Color | Visual color of the portal node for easy identification. |
