# Portal Out


![](../img/portal_out.png)

### Description

Marks an exit point for a hidden connection. Outputs the value received by the referenced **[Portal In](../../../../../content/animations/graph/node_library/portal/portal_in.md)** node. A dotted line is drawn between **[Portal In](../../../../../content/animations/graph/node_library/portal/portal_in.md)** and **Portal Out** nodes when either is selected, showing the hidden connection.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The value received from the referenced **[Portal In](../../../../../content/animations/graph/node_library/portal/portal_in.md)** node. |


## Properties


| Portal | Selects which **Portal In** node this output connects to. |
|---|---|
