# Nand


![](../img/nand.png)

### Description

Returns true unless both **A** and **B** are true (negated **[And](../../../../../content/animations/graph/node_library/comparison/and.md)**).


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/bool.png) | **A** | First operand. |
| ![](../img/types/bool.png) | **B** | Second operand. |
| ![](../img/types/bool.png) | **Result** | true unless both **A** and **B** are true. |
