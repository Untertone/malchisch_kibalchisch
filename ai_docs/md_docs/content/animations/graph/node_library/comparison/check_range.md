# Check Range


![](../img/check_range.png)

### Description

Returns true if **Value** is between **Min** and **Max** (inclusive), false otherwise.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The value to check. |
| ![](../img/types/undefined.png) | **Min** | Lower bound of the range. |
| ![](../img/types/undefined.png) | **Max** | Upper bound of the range. |
| ![](../img/types/bool.png) | **Result** | true if **Value** is within [**Min**, **Max**]. |
