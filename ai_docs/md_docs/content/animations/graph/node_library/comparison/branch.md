# Branch


![](../img/branch.png)

### Description

Outputs one of two values based on a boolean condition. This is equivalent to a ternary operator: if **Condition** is true, outputs the **True** input; otherwise, outputs the **False** input. Only the active branch is evaluated at runtime.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/bool.png) | **Condition** | The condition to evaluate. |
| ![](../img/types/undefined.png) | **True** | Value to output when **Condition** is true. |
| ![](../img/types/undefined.png) | **False** | Value to output when **Condition** is false. |
| ![](../img/types/undefined.png) | **Result** | The selected value. Type matches **True**/**False** inputs. |
