# Not


![](../img/not.png)

### Description

Inverts a boolean value. Returns true if **X** is false, and false if **X** is true.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/bool.png) | **X** | The value to invert. |
| ![](../img/types/bool.png) | **Result** | The inverted boolean value. |
