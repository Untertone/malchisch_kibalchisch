# Sub Graph Inputs


![](../img/sub_graph_inputs.png)

### Description

Defines the input ports of a subgraph. This node appears automatically when you create a new subgraph. Add, remove, rename, and change the type of inputs via the *Inputs* panel on the right side of the editor.


See **[Subgraphs](../../../../../content/animations/sub_graphs/index.md#defining_io)** article for details.


> **Notice:** Available only inside subgraphs.
