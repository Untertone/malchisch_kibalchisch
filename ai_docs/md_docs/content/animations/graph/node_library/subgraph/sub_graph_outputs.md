# Sub Graph Outputs


![](../img/sub_graph_outputs.png)

### Description

Defines the output ports of a subgraph. This node appears automatically when you create a new subgraph. Add, remove, rename, and change the type of outputs via the *Outputs* panel on the right side of the editor.


See **[Subgraphs](../../../../../content/animations/sub_graphs/index.md#defining_io)** article for details.


> **Notice:** Available only inside subgraphs.
