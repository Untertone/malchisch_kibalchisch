# Sub Graph


![](../img/sub_graph.png)

### Description

Embeds a reusable animation graph inside the current graph. The node's input and output ports are determined by the referenced subgraph's **[SubGraph Inputs](../../../../../content/animations/graph/node_library/subgraph/sub_graph_inputs.md)** and **[SubGraph Outputs](../../../../../content/animations/graph/node_library/subgraph/sub_graph_outputs.md)** nodes. When the subgraph is modified, all graphs using it are automatically updated.


Double-click this node to navigate into the subgraph and edit its contents.


See **[Subgraphs](../../../../../content/animations/sub_graphs/index.md)** article for details.


## Properties


| SubGraph | Selects the subgraph to reference. |
|---|---|
