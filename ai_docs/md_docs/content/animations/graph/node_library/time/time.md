# Time


![](../img/time.png)

### Description

Outputs the elapsed time in seconds since the application started. The value increases continuously each frame and is affected by the [time scale](../../../../../editor2/settings/world/index.md).


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Time** | Elapsed time in seconds. |
