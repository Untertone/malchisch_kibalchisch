# State


![](../img/state.png)

### Description

Represents a single state inside a state machine. Each state contains its own animation graph that produces the pose for that state. Double-click a State node to enter its graph and define the animation logic.


One state in each state machine is designated as the **Entry** state - the state that becomes active when the state machine starts.


> **Notice:** Available only inside a **[State Machine](../../../../../content/animations/graph/node_library/state_machine/index.md)**.


## Properties


State name is configured via the [Selected Item](../../../../../content/animations/graph/index.md#side_panels) panel.


| State | Name of the state (e.g., Idle, Walking, Running). |
|---|---|
