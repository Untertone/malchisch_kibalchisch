# State Machine


![](../img/state_machine.png)

### Description

Contains a state machine - a set of states with transitions between them. The node outputs the animation pose from the currently active state (or a blend of two states during a transition). Double-click this node to enter the State Machine graph.


See **[State Machines](../../../../../content/animations/state_machines/index.md)** article for details.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/anim_pose.png) | **Pose** | The pose from the current active state. |


## Properties


| Name | Display name of the state machine. Also used to query this state machine from code. |
|---|---|
