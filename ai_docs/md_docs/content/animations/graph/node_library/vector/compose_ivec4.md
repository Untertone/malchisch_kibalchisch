# Compose Ivec4


![](../img/compose_ivec4.png)

### Description

Assembles an ivec4 from individual integer components **X**, **Y**, **Z**, and **W**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/int.png) | **X** | The X component. |
| ![](../img/types/int.png) | **Y** | The Y component. |
| ![](../img/types/int.png) | **Z** | The Z component. |
| ![](../img/types/int.png) | **W** | The W component. |
| ![](../img/types/ivec4.png) | **Result** | The assembled integer vector. |
