# Compose Ivec2


![](../img/compose_ivec2.png)

### Description

Assembles an ivec2 from individual integer components **X** and **Y**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/int.png) | **X** | The X component. |
| ![](../img/types/int.png) | **Y** | The Y component. |
| ![](../img/types/ivec2.png) | **Result** | The assembled integer vector. |
