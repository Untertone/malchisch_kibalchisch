# Compose Vec3


![](../img/compose_vec3.png)

### Description

Assembles a vec3 from individual float components **X**, **Y**, and **Z**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **X** | The X component. |
| ![](../img/types/float.png) | **Y** | The Y component. |
| ![](../img/types/float.png) | **Z** | The Z component. |
| ![](../img/types/vec3.png) | **Result** | The assembled vector. |
