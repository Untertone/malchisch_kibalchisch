# Compose Vec4


![](../img/compose_vec4.png)

### Description

Assembles a vec4 from individual float components **X**, **Y**, **Z**, and **W**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **X** | The X component. |
| ![](../img/types/float.png) | **Y** | The Y component. |
| ![](../img/types/float.png) | **Z** | The Z component. |
| ![](../img/types/float.png) | **W** | The W component. |
| ![](../img/types/vec4.png) | **Result** | The assembled vector. |
