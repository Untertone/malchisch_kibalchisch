# Angle Between Vectors


![](../img/angle_between_vectors.png)

### Description

Computes the angle between two vectors. Outputs the result via **Radians** and **Degrees** simultaneously.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **A** | First vector. |
| ![](../img/types/undefined.png) | **B** | Second vector. |
| ![](../img/types/float.png) | **Radians** | The angle in radians. |
| ![](../img/types/float.png) | **Degrees** | The angle in degrees. |
