# Length Squared


![](../img/length_squared.png)

### Description

Returns the squared length of **Value**. Faster than **[Length](../../../../../content/animations/graph/node_library/vector/length.md)** (avoids square root), useful when only comparing distances.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The input vector. |
| ![](../img/types/float.png) | **Result** | The squared length of the vector. |
