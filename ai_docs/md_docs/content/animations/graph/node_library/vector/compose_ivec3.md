# Compose Ivec3


![](../img/compose_ivec3.png)

### Description

Assembles an ivec3 from individual integer components **X**, **Y**, and **Z**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/int.png) | **X** | The X component. |
| ![](../img/types/int.png) | **Y** | The Y component. |
| ![](../img/types/int.png) | **Z** | The Z component. |
| ![](../img/types/ivec3.png) | **Result** | The assembled integer vector. |
