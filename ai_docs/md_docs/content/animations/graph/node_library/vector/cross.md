# Cross


![](../img/cross.png)

### Description

Computes the cross product of two 3D vectors. Returns a vector that is perpendicular to both **A** and **B**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/vec3.png) | **A** | First vector. |
| ![](../img/types/vec3.png) | **B** | Second vector. |
| ![](../img/types/vec3.png) | **Result** | The cross product of **A** and **B**. |
