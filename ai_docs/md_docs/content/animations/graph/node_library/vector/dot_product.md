# Dot Product


![](../img/dot_product.png)

### Description

Computes the dot product of two vectors. Useful for measuring alignment between directions.


For example, for normalized vectors:


- 1 - same direction
- 0 - perpendicular
- -1 - opposite direction


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **A** | First vector. |
| ![](../img/types/undefined.png) | **B** | Second vector. |
| ![](../img/types/float.png) | **Result** | The dot product of **A** and **B**. |
