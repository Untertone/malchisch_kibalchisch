# Reflect


![](../img/reflect.png)

### Description

Reflects **V** around a surface defined by **Normal**. Returns the mirror of the input vector with respect to the given normal.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **V** | The vector to reflect. |
| ![](../img/types/undefined.png) | **Normal** | The normal vector to reflect around. |
| ![](../img/types/undefined.png) | **Result** | The reflected vector. |
