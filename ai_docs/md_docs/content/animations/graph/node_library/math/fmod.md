# FMod


![](../img/fmod.png)

### Description

Computes the floating-point remainder of **A** divided by **B**.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **A** | The dividend. |
| ![](../img/types/float.png) | **B** | The divisor. |
| ![](../img/types/float.png) | **Result** | The floating-point remainder of **A** / **B**. |
