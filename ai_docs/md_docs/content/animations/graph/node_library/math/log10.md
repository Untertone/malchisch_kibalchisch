# Base-10 Logarithm

Computes the base-10 logarithm of the input value.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The input value (should be positive). |
| ![](../img/types/float.png) | **Result** | The base-10 logarithm of the input value. |
