# Arccosine


![](../img/acos.png)

### Description

Computes the arccosine of the input value, returning the angle in radians.


- Input range: [-1, 1] (values outside are clamped)
- Result range: [0, PI]


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The value whose arccosine is to be computed. |
| ![](../img/types/float.png) | **Result** | The arccosine in radians. |
