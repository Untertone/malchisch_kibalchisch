# One Minus


![](../img/one_minus.png)

### Description

Returns 1 minus the input value. This is commonly used to invert normalized values.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The input value. |
| ![](../img/types/float.png) | **Result** | 1 minus the input value. |
