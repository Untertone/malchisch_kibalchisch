# Square Root


![](../img/sqrt.png)

### Description

Computes the square root of the input value.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The input value (should be non-negative). |
| ![](../img/types/float.png) | **Result** | The square root of the input value. |
