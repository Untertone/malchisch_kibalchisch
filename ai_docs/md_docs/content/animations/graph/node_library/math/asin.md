# Arcsine


![](../img/asin.png)

### Description

Computes the arcsine of the input value, returning the angle in radians.


- Input range: [-1, 1] (values outside are clamped)
- Result range: [-PI/2, PI/2]


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The value whose arcsine is to be computed. |
| ![](../img/types/float.png) | **Result** | The arcsine in radians. |
