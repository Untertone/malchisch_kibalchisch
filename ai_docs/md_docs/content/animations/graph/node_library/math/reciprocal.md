# Reciprocal


![](../img/reciprocal.png)

### Description

Returns 1 divided by the input value (1 / **Value**).


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The input value (should be non-zero). |
| ![](../img/types/float.png) | **Result** | The reciprocal of the input value. |
