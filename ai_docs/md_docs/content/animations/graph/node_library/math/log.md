# Base-E Logarithm


![](../img/log.png)

### Description

Computes the natural logarithm (base e) of the input value.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The input value (should be positive). |
| ![](../img/types/float.png) | **Result** | The natural logarithm of the input value. |
