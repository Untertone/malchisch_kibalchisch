# Ceiling


![](../img/ceil.png)

### Description

Returns the smallest integer that is greater than or equal to the input value.


- Ceil(4.2) = 5
- Ceil(-2.5) = -2


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The input value. |
| ![](../img/types/undefined.png) | **Result** | The input value rounded up. |
