# Base-2 Logarithm


![](../img/log2.png)

### Description

Computes the base-2 logarithm of the input value.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The input value (should be positive). |
| ![](../img/types/float.png) | **Result** | The base-2 logarithm of the input value. |
