# Step


![](../img/step.png)

### Description

Returns 0 if **A** is less than **B**, and 1 otherwise. This is useful for creating hard threshold transitions.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **A** | The value to test. |
| ![](../img/types/undefined.png) | **B** | The threshold edge value. |
| ![](../img/types/undefined.png) | **Result** | 0 if **A** < **B**, otherwise 1. |
