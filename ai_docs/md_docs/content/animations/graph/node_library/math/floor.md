# Floor


![](../img/floor.png)

### Description

Returns the largest integer that is less than or equal to the input value.


- Floor(4.2) = 4
- Floor(-2.5) = -3


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The input value. |
| ![](../img/types/undefined.png) | **Result** | The input value rounded down. |
