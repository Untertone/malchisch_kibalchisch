# Sign


![](../img/sign.png)

### Description

Returns the sign of the input value:


- Negative returns -1
- Zero returns 0
- Positive returns 1


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The input value. |
| ![](../img/types/undefined.png) | **Result** | -1 if the value is negative, 0 if zero, or 1 if positive. |
