# Clamp


![](../img/clamp.png)

### Description

Constrains **Value** to lie between **Min** and **Max**:


- If **Value** < **Min**, returns **Min**
- If **Value** > **Max**, returns **Max**
- Otherwise, returns **Value** unchanged


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/undefined.png) | **Value** | The value to clamp. |
| ![](../img/types/undefined.png) | **Min** | The lower bound. |
| ![](../img/types/undefined.png) | **Max** | The upper bound. |
| ![](../img/types/undefined.png) | **Result** | The clamped value. |
