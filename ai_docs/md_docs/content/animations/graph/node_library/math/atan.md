# Arctangent


![](../img/atan.png)

### Description

Computes the arctangent of the input value, returning the angle in radians.


- Result range: [-PI/2, PI/2]


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Value** | The value whose arctangent is to be computed. |
| ![](../img/types/float.png) | **Result** | The arctangent in radians. |
