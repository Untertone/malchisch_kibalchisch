# Landscape


![](../../../samples/img/art_samples_landscape.jpg)

The Landscape content sample contains a set of scenes demonstrating the following:


- [Landscape](../../../objects/objects/terrain/landscape_terrain/index.md) with high-resolution terrain insets using [landscape layer maps](../../../objects/objects/terrain/landscape_terrain/landscape_layer_map.md), [blending modes](../../../objects/objects/terrain/landscape_terrain/landscape_layer_map.md#import_parameters)
- Using masks of [*Landscape Terrain* details](../../../objects/objects/terrain/landscape_terrain/details.md): Blending by Heightmap using various *[Contrast](../../../content/materials/library/landscape_terrain_detail_base/index.md#base)* values for transition between details, *[Additional Mask](../../../content/materials/library/landscape_terrain_detail_base/index.md#additional_mask)*, and *[Triplanar Mapping](../../../content/materials/library/landscape_terrain_detail_base/index.md#base)*, adjusting visibility distance for details, setting the angle and offset for the texture
- Creating a tunnel cutting through the landscape using [decal-based holes](../../../objects/objects/terrain/landscape_terrain/index.md#decal_holes) with the [decal_terrain_hole_base](../../../content/materials/library/decal_terrain_hole_base/index.md) material assigned
- Example airfield created using the *Landscape Terrain, Landscape Layer Maps* and [Mesh Decals](../../../objects/decals/mesh/index.md)
- Example with wheel traces and terrain geometry deformation
- Example imitating night lights using the [decal emission state](../../../content/materials/library/decal_base/index.md#option_emission)
