# Particles


![](../../../samples/img/art_samples_particles.jpg)

The *Particles* content sample contains a set of scenes demonstrating the following:


- [Particle Systems](../../../objects/effects/particles/index.md) material settings
- Particle Systems parameters
- Using several Particle Systems to create a composite effect
- [Types of particles](../../../objects/effects/particles/index.md#particle_type)
- [Shapes of emitters](../../../objects/effects/particles/index.md#emitter_shape)
- Setting such parameters as color, direction, radius, and position using [curves](../../../editor2/curve_editor/index.md)
- Additional [forces](../../../objects/effects/particles/index.md#forces) and [physicals](../../../objects/effects/physicals/index.md) affecting particles
- Settings available in the [particles material](../../../content/materials/library/particles_base/index.md)
- [Rendering to a procedural texture](../../../objects/effects/particles/index.md#procedural_rendering) used by the [orthographic decal](../../../objects/decals/ortho/index.md)
- Use of particle systems with the [skinned mesh](../../../objects/objects/mesh_skinned/index.md)
- Rain effect using particles
