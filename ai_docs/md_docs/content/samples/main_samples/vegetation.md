# Vegetation


![](../../../samples/img/art_samples_vegetation.jpg)

The Vegetation content sample contains a set of scenes demonstrating the following:


- Forest example
- [Grass](../../../objects/objects/grass/index.md) parameters, materials, and animation settings
- Using masks and fields with Grass
- Orienting Grass along the terrain
- Optimizing grass using its visibility, distance fade, and thinning parameters
- [Types of vegetation animation](../../../content/materials/library/mesh_base/index.md#vegetation_leaves)
- [Vertex- and UV-based animation of leaves, branches, and trunks](../../../content/materials/library/mesh_base/index.md#vegetation_animation)
- Tree animation sample
