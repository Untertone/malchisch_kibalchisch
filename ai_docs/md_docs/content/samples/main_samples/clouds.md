# Clouds


![](../../../samples/img/art_samples_clouds.jpg)

The Clouds content sample contains a set of scenes with the [Cloud Layer](../../../objects/objects/cloud_layer/index.md) that demonstrate the following:


- Cloud material settings
- Using *[Field Weather](../../../objects/effects/fields/field_weather/index.md)*
- Various types and forms of clouds, both built-in and created using [custom 3D textures](../../../content/materials/library/clouds_base/index.md#textures)
- [Sorting](../../../editor2/settings/render_settings/clouds/index.md#accurate_layers_sorting) of several cloud layers
