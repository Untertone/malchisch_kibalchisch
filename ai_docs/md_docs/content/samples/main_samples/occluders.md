# Occluders


![](../../../samples/img/art_samples_occluders.jpg)

The Occluders content sample demonstrates culling of objects by [Occluders](../../../objects/worlds/world_occluders/index.md). It contains a simplified model of a house that also makes it possible to assess culling of outdoor objects when the camera is inside the building.
