# Reflections


![](../../../samples/img/art_samples_reflections.jpg)

The Reflections content sample contains a set of scenes demonstrating the following:


- Shapes of [Environment Probes](../../../objects/lights/envprobe/index.md), types of reflections
- *Environment probe* settings
- Using the [reflection mask](../../../content/optimization/reflections/index.md#reflection_mask)
- Clipping occluded reflections using the *[Cutout By Shadow](../../../objects/lights/envprobe/index.md#cutout_by_shadow)* option
- *[Screen Space Reflection (SSR)](../../../editor2/settings/render_settings/ssr/index.md)* render settings and usage
