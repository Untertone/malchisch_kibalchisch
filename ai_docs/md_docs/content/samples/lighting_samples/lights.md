# Lights


![](../../../samples/img/art_samples_lights.jpg)

The *Lights* content sample contains a set of scenes demonstrating the following:


- [Omni Light](../../../objects/lights/omni/index.md) settings: attenuation, penumbra, softness, bias, various shapes of the light source, modulation using IES profiles
- [Projected Light](../../../objects/lights/proj/index.md) settings: field of view, attenuation, attenuation penumbra, shadow, shadow penumbra, softness, bias, various shapes of the light source, modulation by simple textures and IES profiles
- [Static and dynamic shadows](../../../content/optimization/lights/index.md#static_light), screen space shadows
- [Shadow mask](../../../principles/bit_masking/index.md#shadow_mask)
- [Shadow penumbra](../../../editor2/settings/render_settings/shadows/index.md#penumbra_mode)
- Imitation of baked light from light posts using [lightmapping](../../../editor2/lighting/gi/lightmaps.md) and [emissive](../../../content/materials/library/mesh_base/index.md#option_emission) objects
