# Lighting


![Lighting](index.png)


The set of samples in this part of *Art Samples* contains scenes demonstrating the following:


- Available light objects and their settings
- Various types of shadows
- Emission
- Use of [Voxel Probes](../../../objects/lights/voxelprobe/index.md) and [lightmapping](../../../editor2/lighting/gi/lightmaps.md)
- Types of reflections and their implementation
