# Global Illumination


![](../../../samples/img/art_samples_global_illumination.jpg)

The *Global Illumination* content sample contains a set of scenes demonstrating the use of [Voxel Probes](../../../objects/lights/voxelprobe/index.md) and [lightmapping](../../../editor2/lighting/gi/lightmaps.md):


- Examples of baking from various emissive objects
- Cornell box
- Optimized placement and usage
- Additive blending
- Recommended use cases for Voxel Probes and lightmaps
- Setting lightmaps for LODs
- Example of a simple interior scene demonstrating the illumination settings
- Scene containing the voxel probe and a number of dynamic objects that are [affected by the change of the sun angle](../../../objects/lights/voxelprobe/index.md#sun_color)
