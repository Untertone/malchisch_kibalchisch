# Ice Sample


| [**View Fullscreen**](https://matgraph.unigine.com/SampleIce_2.21/fullView) |
|---|


This material graph sample demonstrates how to create an effect of a multi-layered ice material.


The sample showcases the use of the **[Parallax Occlusion Mapping](../../../../../content/materials/graph/node_library/misc/parallax_occlusion_mapping.md)** node providing displaced UVs for the *Albedo* texture based on the input heightmap.


The effect is augmented with roughness irregularities taken from additional textures and internal emission based on height.
