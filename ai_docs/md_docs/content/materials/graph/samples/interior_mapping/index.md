# Interior Mapping Sample


This material graph sample demonstrates how to create the effect of parallax interior mapping when creating materials.


![](result.jpg)

*The result*


The world contains two material graphs for different approaches and the source interior used to grab the textures.


## Cube Mapped Interior


| [**View Fullscreen**](https://matgraph.unigine.com/SampleInteriorMappingCubemap_2.21/fullView) |
|---|


The `interior_mapping_cubemap` material graph showcases the use of the **[Interior Mapping Cubemap](../../../../../content/materials/graph/node_library/misc/interior_mapping_cubemap.md)** node for interior mapping based on cubemap textures.


## 2D Texture Mapping


| [**View Fullscreen**](https://matgraph.unigine.com/SampleInteriorMappingTexture2d_2.21/fullView) |
|---|


The `interior_mapping_texture_2d` material graph showcases the use of the **[Interior Mapping Texture 2D](../../../../../content/materials/graph/node_library/misc/interior_mapping_texture_2d.md)** node for interior mapping based on 2D textures having the fixed layout that can be adjusted via a set of parameters.
