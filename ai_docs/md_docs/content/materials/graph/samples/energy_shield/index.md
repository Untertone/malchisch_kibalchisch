# Energy Shield Sample


| [**View Fullscreen**](https://matgraph.unigine.com/SampleEnergyShield_2.21/fullView) |
|---|


This material graph sample demonstrates how to implement an effect of an animated energy shield.


This material is an extended version of the **[Fade By Depth](../../../../../content/materials/graph/samples/fade_by_depth/index.md)** sample. Along with the Fresnel effect and depth fading, opacity is also controlled by a set of textures adding a hexagon pattern and noise to it. An additional gradient mask is used to add a scan line effect animated with the time.


The effect also provides time-based geometry inflation by controlling the vertex offset in the tangent space.
