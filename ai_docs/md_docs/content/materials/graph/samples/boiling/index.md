# Boiling Sample


| [**View Fullscreen**](https://matgraph.unigine.com/SampleBoiling_2.21/fullView) |
|---|


This material graph sample demonstrates creation of a complex effect of boiling liquid featuring tessellation and vertex offset control.


The material graph demonstrates how to create a mask changing with the time using an offset texture (*bubbles_offset*) and control offset of tessellated geometry and emission based on the mask.
