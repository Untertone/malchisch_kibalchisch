# Melting Sample


| [**View Fullscreen**](https://matgraph.unigine.com/SampleMelting_2.21/fullView) |
|---|


This material graph sample demonstrates how to create an effect of a melting geometry.


The material graph utilizes a 3D noise texture defining the irregularities of height-based vertex shift. A set of parameters defines the emission intensity and fade distance of the effect.
