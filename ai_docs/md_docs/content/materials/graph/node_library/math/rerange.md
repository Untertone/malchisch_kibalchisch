# Rerange Node


![](../img/rerange.png)

### Description

Outputs an input value In set within a source range [In Range Min, In Range Max] converted to a new one within a target range [Out Range Min, Out Range Max].


This node is especially useful when you need to change gradient's width (make it wider or narrower), shift, or even invert it.
