# Combobox Switch Node


![](../img/combobox_switch.png)

### Description

This node is an analog of the **[Branch](../../../../../content/materials/graph/node_library/logical/branch.md)**, but with multiple options instead of only two. Connect a **[Combobox](../../../../../content/materials/graph/node_library/params/combobox.md)** parameter node to the input. As a result only the branch that corresponds to a condition selected in the **[Combobox](../../../../../content/materials/graph/node_library/params/combobox.md)** parameter will be evaluated. Thus, you can partially change material's functionality and define names for different modes.
