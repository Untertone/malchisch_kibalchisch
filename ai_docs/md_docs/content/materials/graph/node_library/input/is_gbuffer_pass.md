# Is GBuffer Pass Node


![](../img/is_gbuffer_pass.png)

### Description

Outputs a boolean value indicating that [*GBuffer* pass](../../../../../principles/render/sequence/index.md#gbuffer) is currently in progress. You can pass this value to a [Branch](../../../../../content/materials/graph/node_library/logical/branch.md) node to change the look of your material during the *GBuffer* pass.
