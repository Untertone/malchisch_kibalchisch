# Is Auxiliary Pass Node


![](../img/is_auxiliary_pass.png)

### Description

Outputs a boolean value indicating that [*Auxiliary* rendering pass](../../../../../principles/render/sequence/index.md#auxiliary) is currently in progress. You can pass this value to a [Branch](../../../../../content/materials/graph/node_library/logical/branch.md) node to change the look of your material during the *Auxiliary* pass.
