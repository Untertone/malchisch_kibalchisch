# Camera Direction Node


![](../img/camera_direction.png)

### Description

Provides access to the inverted normalized Camera Direction Vector. Unlike the [View Direction Vector](../../../../../content/materials/graph/node_library/input/view_direction.md) it has the same value for all pixels on the screen. The coordinate space of the output value (*World, Object, Tangent, View*) can be selected with the Space dropdown parameter (double-click somewhere inside the node to see the dropdown).
