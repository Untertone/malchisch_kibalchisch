# Sun Direction Node


![](../img/sun_direction.png)

### Description

Provides access to the direction vector of the [**World**](../../../../../objects/lights/world/index.md) light source having the [**Sun**](../../../../../objects/lights/world/index.md#light_settings) scattering mode and currently used in the scene. If the scene doesn't have such a light source enabled a default vector pointing upwards is returned.


The coordinate space of the output value (*World, Object, Tangent, View*) can be selected with the Space dropdown parameter (double-click somewhere inside the node to see the dropdown).
