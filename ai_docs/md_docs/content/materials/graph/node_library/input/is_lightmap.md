# Is Lightmap Node


![](../img/is_lightmap.png)

### Description

Outputs a boolean value indicating that [lightmaps](../../../../../editor2/lighting/gi/lightmaps.md) are used for the surface to which the material is assigned. You can pass this value to a [Branch](../../../../../content/materials/graph/node_library/logical/branch.md) node to change the look of your material when lightmaps are used.
