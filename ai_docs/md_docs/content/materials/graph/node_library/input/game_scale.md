# Game Scale Node


![](../img/game_scale.png)

### Description

Outputs the value that is used to scale frame duration. This value affects both the rendering rate (for example, particles spawn rate and material offsets) and [physics ticks](../../../../../editor2/settings/physics_global/index.md#physics_scale).


This node scales the value output by the [IFps](../../../../../content/materials/graph/node_library/input/ifps.md) node.
