# Vertex Normal Node


![](../img/vertex_normal.png)

### Description

Provides access to the Normal Vector of initial geometry. This vector is calculated based on UV 0. The coordinate space of the output value (*World, Object, Tangent, View*) can be selected with the Space dropdown parameter (double-click somewhere inside the node to see the dropdown).
