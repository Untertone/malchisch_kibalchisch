# Fibonacci Hemisphere Node


![](../img/fibonacci_hemisphere.png)

### Description

Generates a set of points with X, Y and Z coordinates in the [-1; 1] range that describe uniform distribution of samples on the surface of a unit hemisphere.


- Index - index of the current point for the hemisphere with the number of points equal to Samples
- Samples - number of points
- Noise Intensity - jittering of samples distribution
- Noise Pattern - the noise pattern (animated blue noise by default)


To get uniform distribution of coordinates for a hemisphere, use the Fibonacci Hemisphere node.
