# Hammersley Node


![](../img/hammersley.png)

### Description

Outputs X and Y coordinates of a point with the Index index in a Hammersley Point Set that describes uniform distribution of points in the [0; 1] range.


- Index - index of the current point int the set with the number of points equal to Samples Number.
- Samples Number - number of points.


If you want to get uniform distribution of coordinates for a square, the Hammersley node fits perfectly.
