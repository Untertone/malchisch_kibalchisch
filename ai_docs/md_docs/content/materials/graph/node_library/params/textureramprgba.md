# TextureRamp RGBA Parameter Node


![](../img/texture_ramp_rgba_param.png)

### Description

The **TextureRamp RGBA** parameter parameter enables you to adjust a four-component 2D curve in the form of an RGBA-ramp texture.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as a ramp widget enabling you to adjust a four-component 2D curve.
