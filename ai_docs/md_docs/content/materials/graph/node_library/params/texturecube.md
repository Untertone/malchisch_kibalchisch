# TextureCube Parameter Node


![](../img/texturecube_param.png)

### Description

The **TextureCube** parameter enables you to choose a cubemap texture (an array of 2D textures with 6 elements for each face of the cube).


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as an asset widget enabling you to choose a cubemap texture asset from the data directory.
