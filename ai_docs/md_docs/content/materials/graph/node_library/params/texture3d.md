# Texture3D Parameter Node


![](../img/texture3d_param.png)

### Description

The **Texture3D** parameter enables you to bring objects of your project to life. Here volume textures with 3D texels are used to lie over or wrap around objects.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as an asset widget enabling you to choose a texture asset from the data directory.
