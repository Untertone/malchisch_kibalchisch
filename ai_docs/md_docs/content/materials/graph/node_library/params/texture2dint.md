# Texture2DInt Parameter Node


![](../img/texture2dint_param.png)

### Description

The **Texture2DInt** parameter enables you to bring objects of your project to life. Here two-dimensional (2D) images with **int** elements are used to lie over or wrap around objects.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as an asset widget enabling you to choose a texture asset from the data directory.
