# TextureRamp RG Parameter Node


![](../img/texture_ramp_rg_param.png)

### Description

The **TextureRamp RG** parameter enables you to adjust a two-component 2D curve in the form of an RG-ramp texture.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as a ramp widget enabling you to adjust a two-component 2D curve.
