# TextureRamp RGB Parameter Node


![](../img/texture_ramp_rgb_param.png)

### Description

The **TextureRamp RGB** parameter enables you to adjust a three-component 2D curve in the form of an RGB-ramp texture.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as a ramp widget enabling you to adjust a three-component 2D curve.
