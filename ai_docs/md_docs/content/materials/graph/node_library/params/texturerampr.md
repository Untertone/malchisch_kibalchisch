# TextureRamp R Parameter Node


![](../img/texture_ramp_r_param.png)

### Description

The **TextureRamp R** parameter enables you to adjust a single-component 2D curve in the form of an R-ramp texture.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as a ramp widget enabling you to adjust a single-component 2D curve.
