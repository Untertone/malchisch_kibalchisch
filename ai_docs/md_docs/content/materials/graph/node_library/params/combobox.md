# Combobox Parameter Node


![](../img/combobox_param.png)

### Description

The **ComboBox** parameter node is to be connected to the **[Combobox Switch](../../../../../content/materials/graph/node_library/logical/combobox_switch.md)** node and define the number and the names of its inputs. When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as a *ComboBox* with a list of options.


Use the fields in the *Parameters* tab to define the number and the items of this list.
