# Bool Parameter Node


![](../img/bool_param.png)

### Description

The **bool** parameter enables you to toggle different features in your material on and off (e.g., for performance optimization), when you connect it to the *Condition* input of the **[Branch](../../../../../content/materials/graph/node_library/logical/branch.md)** node. For any mode of the **[Branch](../../../../../content/materials/graph/node_library/logical/branch.md)** node (*Flatten, Branch*, or *Auto*), only the branch connected to the *True* input is evaluated.


When selecting the material in UnigineEditor you'll see this parameter in the *Parameters* window as a CheckBox.
