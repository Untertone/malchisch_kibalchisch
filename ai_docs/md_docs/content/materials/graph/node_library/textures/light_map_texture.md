# Light Map Texture Node


![](../img/light_map_texture.png)

### Description

This node represents a 2D light map texture, that is assigned to an object's surface in the *[Texture](../../../../../editor2/node_parameters/visual_representation/index.md#lightmaps_texture)* field of the *Lightmaps* group.


The advantage of this texture is that you can assign the same material to different objects and use different texture inside the material as each surface can have a unique lightmap.
