# Perspective Node


![](../img/perspective.png)

### Description

Outputs the perspective projection matrix.


![](../../../../../api/library/math/perspective_desc.png)


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float.png) | **Fov** | Field of view angle |
| ![](../img/types/float.png) | **Aspect** | Aspect ratio (the ratio of width to height) |
| ![](../img/types/float.png) | **N** | Distance to the near depth clipping plane |
| ![](../img/types/float.png) | **F** | Distance to the farther depth clipping plane |
