# Inverse Transform Node


![](../img/inverse_transform.png)

### Description

Inverts a matrix that consists of a 3x4 sub-matrix (upper left) and a translation vector. The last row of the matrix is ignored. Compared to the [**Inverse**](../../../../../content/materials/graph/node_library/matrix/inverse.md) node, this one is a bit faster and, which is more important, more stable.
