# Refract Node


![](../img/refract.png)

### Description

Returns the refraction vector for the given **Ray Direction** vector, surface defined by the **Normal** vector and the **Refraction Index**.


The *Refraction Index* is the ratio of indices of refraction for two transparent environments. If you want to get a refraction vector for a material with IOR 1.6, pass *1 / 1.6* value to the input.
