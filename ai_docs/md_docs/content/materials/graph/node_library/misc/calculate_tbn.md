# Calculate TBN Node


![](../img/calculate_tbn.png)

### Description

Outputs the calculated Tangent, Binormal, and Normal vectors based on the Normal vector, Position vector, and UV.


#### Ports

| Name | Description |  |
|---|---|---|
| ![](../img/types/float3.png) | **N** | Normal vector |
| ![](../img/types/float3.png) | **Position** | Position vector |
| ![](../img/types/float2.png) | **UV** | UV |
| ![](../img/types/float3.png) | **T** | Tangent |
| ![](../img/types/float3.png) | **B** | Binormal |
| ![](../img/types/float3.png) | **N** | Normal vector |
