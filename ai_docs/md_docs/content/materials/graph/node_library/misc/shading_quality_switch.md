# Shading Quality Switch Node


![](../img/shading_quality_switch.png)

### Description

Receives different sets of shading features for different quality levels (**Low**, **Medium**, **High**) and outputs the corresponding implementation depending on the [quality level selected globally](../../../../../editor2/settings/render_settings/shading_quality/index.md). It allows you more flexibility in maintaining quality-performance balance in your projects. Simply add the **Shading Quality Switch** node when building your graph and connect different implementation options to the corresponding ports. Different sets of shading features will be generated automatically and used for the corresponding selected quality level.
