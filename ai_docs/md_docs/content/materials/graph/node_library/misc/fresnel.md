# Fresnel Node


![](../img/fresnel.png)

### Description

This node calculates a falloff based on the viewing angle at which the surface is viewed. The output value can be used as the intensity for *Fresnel reflections*, transparency, and other effects.


For usage examples, refer to the [**Fresnel Effect**](../../../../../content/materials/graph/samples/fresnel_effect/index.md) material graph sample.
