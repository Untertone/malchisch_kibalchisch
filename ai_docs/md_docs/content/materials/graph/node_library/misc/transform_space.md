# Transform Space Node


![](../img/transform_space.png)

### Description

This node is used to transform a **POSITION** vector from one space (*Camera World, Object, View, Absolute World*) to another space. Suppose you have a position coordinate in *Object* space and you want to use it where the same position coordinate is required but in a different space - *View*, for example. You can use this node to switch to the required space.


> **Notice:** Double-click somewhere within a node to change initial (*From*) and target (*To*) space.
