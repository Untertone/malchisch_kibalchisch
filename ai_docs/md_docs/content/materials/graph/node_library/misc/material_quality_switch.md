# Material Quality Switch Node


![](../img/material_quality_switch.png)

### Description

Receives different implementations of the same material for different quality levels (**Low**, **Medium**, **High**) and outputs the corresponding implementation depending on the [quality level selected globally](../../../../../editor2/settings/render_settings/materials_quality/index.md). It allows you more flexibility in maintaining quality-performance balance in your projects. Simply add the **Material Quality Switch** node when building your graph and connect different implementation options to the corresponding ports. Different versions of materials will be generated automatically and used for the corresponding selected quality level.
