# Flipbook Node


![](../img/flipbook.png)

### Description

Remaps the input UV coordinates to a region of a grid layout having X Axis Tiles columns and Y Axis Tiles rows.


- X Axis Tiles - number of columns
- Y Axis Tiles - number of rows
- Tile - index of the current region
- UV - UV coordinates to be modified. [**Vertex UV 0**](../../../../../content/materials/graph/node_library/input/vertex_uv_0.md) coordinates are used by default.


You can use the Flipbook node for impostors or sprite animation.
