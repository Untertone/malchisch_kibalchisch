# GUI


A **GUI** object is a flat or mesh-based graphical element, which can have various [widgets](../../../code/gui/index.md) assigned to it. For *GUI* objects the *[gui_base](../../../content/materials/library/gui_base/index.md)* material is used.


There are two types of *GUI* objects:


- [![](gui.png)](../../../objects/objects/gui/gui_object.md) **[GUI](../../../objects/objects/gui/gui_object.md)** is a flat *GUI* object.
- [![](gui.png)](../../../objects/objects/gui/gui_mesh.md) **[GUI Mesh](../../../objects/objects/gui/gui_mesh.md)** is a non-flat *GUI* object based on an arbitrary mesh.


> **Notice:** The *GUI* object is added to the world as a node and can be managed via UnigineEditor.
