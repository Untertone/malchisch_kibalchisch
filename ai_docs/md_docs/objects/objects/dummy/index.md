# Dummy Object


A ![](dummy.png) **Dummy Object** is an object that is not present in the world visually but shares all properties that other objects have. It is similar to the *[Dummy Node](../../../objects/nodes/dummy/index.md)* except for the possibility of assigning a [physical body](../../../principles/physics/bodies/index.md) to it.


It is usually used as an invisible object with physics properties and can also serve as a root node to organize objects in a hierarchy. Also, for example, it can be used as a base object to hold visible physical object fixed by using [joints](../../../principles/physics/joints/index.md).


### See Also


- The *[ObjectDummy](../../../api/library/objects/class.objectdummy_cpp.md)* class to edit the *Dummy Object* via API


## Adding a Dummy Object


To add a *Dummy Object* to the scene via UnigineEditor do the following:


1. [Run](../../../editor2/index.md#run) UnigineEditor.
2. On the Menu bar, click *Create -> Physics -> Object Dummy*. ![](dummy_create.png)
3. Place the *Dummy Object* somewhere in the world.
