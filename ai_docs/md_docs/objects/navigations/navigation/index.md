# Navigation Areas


A **navigation area** is an area within which pathfinding is performed.


Navigation areas can be the following:


- **[![](sector.png)](../../../objects/navigations/navigation/navigation_sector/index.md)   [Navigation Sector](../../../objects/navigations/navigation/navigation_sector/index.md)** is a cuboid-shaped navigation area within which 2D and 3D routes can be calculated. > **Warning:** 3D navigation feature is experimental and not recommended for production use.
- **[![](mesh.png)](../../../objects/navigations/navigation/navigation_mesh/index.md)   [Navigation Mesh](../../../objects/navigations/navigation/navigation_mesh/index.md)** is a navigation area which is based on an arbitrary mesh. Only 2D routes can be calculated within this navigation area.
