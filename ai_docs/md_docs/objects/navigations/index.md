# Pathfinding Objects


**Pathfinding objects** are the objects that are involved in the pathfinding process.


There are 2 types of pathfinding objects:


- **[![](navigation.png)](../../objects/navigations/navigation/index.md)   [Navigation Area](../../objects/navigations/navigation/index.md)** is a navigation area within which pathfinding is performed.
- **[![](obstacle.png)](../../objects/navigations/obstacle/index.md)   [Obstacle](../../objects/navigations/obstacle/index.md)** is an object that is detected and bypassed by other objects during pathfinding.
