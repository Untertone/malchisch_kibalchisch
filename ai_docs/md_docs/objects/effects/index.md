# Effects


A set of special effects:


- [![](particles.png)](../../objects/effects/particles/index.md)   **[Particle system](../../objects/effects/particles/index.md)** is a complex object consisting of an emitter, emitted particles and a force applied. It is used to simulate effects of fire, smoke, explosions, precipitations, electricity, fountains etc.
- [![](volumes.png)](../../objects/effects/volumetrics/index.md)   **[Volumetric objects](../../objects/effects/volumetrics/index.md)** are used to simulate different types of volumetric effects such as fog, clouds, light beams, burning halos etc.
- [![](fields.png)](../../objects/effects/fields/index.md)   **[Fields](../../objects/effects/fields/index.md)** are cuboid or ellipsoid shaped objects that influence only the selected areas of grass and water geometry. They can either specify areas that should not be rendered or animate the selected geometry areas.
- [![](physicals.png)](../../objects/effects/physicals/index.md)   **[Physicals](../../objects/effects/physicals/index.md)** are objects inside of which there is an invisible force field affecting other objects.
