# Fields


A **field** is a cuboid or ellipsoid shaped object that influences only the selected areas of grass and water geometry. It can either specify areas that should not be rendered or animate the selected geometry areas.


### Types of Fields


There are five types of fields available:


- [![](spacer.png)](../../../objects/effects/fields/field_spacer/index.md)   **[Field Spacer](../../../objects/effects/fields/field_spacer/index.md)** is an object, specifying geometry areas of grass and water that should not be rendered.
- [![](animation.png)](../../../objects/effects/fields/field_animation/index.md)   **[Field Animation](../../../objects/effects/fields/field_animation/index.md)** is an object, specifying geometry areas of grass and vegetation that should be animated.
- [![](height.png)](../../../objects/effects/fields/field_spacer/index.md)   **[Field Height](../../../objects/effects/fields/field_height/index.md)** is an object, specifying global water areas where height should be changed (for wakefields, water disturbance, etc.).
- [![](shoreline.png)](../../../objects/effects/fields/field_shoreline/index.md)   **[Field Shoreline](../../../objects/effects/fields/field_shoreline/index.md)** is an object, specifying global water areas where shoreline swash zone should be located.
- [![](weather.png)](../../../objects/effects/fields/field_weather/index.md)   **[Field Weather](../../../objects/effects/fields/field_weather/index.md)** is an object, specifying a local cloud coverage area.
