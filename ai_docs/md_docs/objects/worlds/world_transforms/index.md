# Transform


**Transform** is a succession of transformations.


### Types of Transforms


There are two types of *Transform*:


- ![](transform.png) **[Transform Path](../../../objects/worlds/world_transforms/transform_path/index.md)** — a succession of transformations based on an arbitrary path
- ![](transform.png) **[Transform Joint](../../../objects/worlds/world_transforms/transform_bone/index.md)** — a succession of transformations based on the jpint animation. It should be a child of *[Mesh Skinned](../../../objects/objects/mesh_skinned_legacy/index.md)* and have one joint assigned.
