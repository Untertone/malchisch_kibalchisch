## geodetics_transformer_00


Demonstrates geo-coordinates conversion using the GeodeticsTransformer class implemented in the UnigineGeodetics plugin. The following maps were used in the sample: World Map Mercator (marked as public domain, license details: <a href='https://creativecommons.org/licenses/by-sa/3.0/legalcode'>CC BY-SA 3.0</a> ) and South Pole Stereographic EPSG 3031 (courtesy of Quantarctica and the Norwegian Polar Institute).
