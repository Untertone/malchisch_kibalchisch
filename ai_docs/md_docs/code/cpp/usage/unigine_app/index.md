# С++ Integration Samples


The engine can be integrated into other systems by using the [CustomSystemProxy](../../../../code/cpp/usage/unigine_app/proxy.md) class.


UNIGINE SDK includes an example of such integration into the Qt framework (`SystemProxyQt.h` and `SystemProxyQt.cpp` in the `source/apps/main_qt` folder).
