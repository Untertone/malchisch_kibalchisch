# Auxiliary Types


The list of auxiliary types of values that are used in some Math functions.


| Type | Definition | Description |
|---|---|---|
| **mat2x2_values** | float[4] | The auxiliary type to store the [2x2 matrix](../../../api/library/math/class.mat2_cpp.md) values. Values are stored column-major. |
| **mat3x3_values** | float[9] | The auxiliary type to store the [3x3 matrix](../../../api/library/math/class.mat3_cpp.md) values. Values are stored column-major. |
| **mat4x3_values** | float[12] | The auxiliary type to store the 4x3 matrix values. Values are stored column-major. |
| **mat4x4_values** | float[16] | The auxiliary type to store the [4x4 matrix](../../../api/library/math/class.mat4_cpp.md) values. Values are stored column-major. |
| **dmat4x3_values** | double[12] | The auxiliary type to store the 4x3 matrix double values. Values are stored column-major. |
| **dmat4x4_values** | double[16] | The auxiliary type to store the 4x4 matrix double values. Values are stored column-major. |
