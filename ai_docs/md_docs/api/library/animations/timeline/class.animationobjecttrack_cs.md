# Unigine::AnimationObjectTrack Class (CS)

**Inherits from:** AnimationObject


This class enables you to manage proxy [animation objects](../../../../principles/animations/index.md#animation_object) used to control animation tracks. Binding to a specific animation track in the framework of a specific [playback](../../../../principles/animations/index.md#animation_playback) is performed by the track name.
