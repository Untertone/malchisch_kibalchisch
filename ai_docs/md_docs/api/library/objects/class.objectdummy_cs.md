# ObjectDummy Class (CS)

**Inherits from:** Object


A [dummy object](../../../objects/objects/dummy/index.md) can be used when an invisible object with physical properties is required. For example, as a base object to hold visible physical object fixed by using [joints](../../../api/library/physics/class.joint_cs.md). It can also serve a root node to organize hierarchy same as [NodeDummy](../../../api/library/nodes/class.nodedummy_cs.md).
