# Unigine::BodyDummy Class (CS)

**Inherits from:** Body


This class is used to create a dummy body that serves as unmoving, non-interacting (and often [invisible](../../../api/library/objects/class.objectdummy_cs.md)) prop to attach other bodies with joints to.
