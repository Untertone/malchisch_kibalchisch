# Unigine::Timer Struct (CPP)

**Header:** #include <UnigineTimer.h>


This structure implements the timer functionality.


## Timer Class

### Members

---

## void begin ( )

Starts the timer.
## long long end ( )

Returns the elapsed time in microseconds and stops the timer.
### Return value

Elapsed time in microseconds.
## double endSeconds ( )

Returns the elapsed time in seconds and stops the timer.
### Return value

Elapsed time in seconds.
## double endMilliseconds ( )

Returns the elapsed time in milliseconds and stops the timer.
### Return value

Elapsed time in milliseconds.
## long long getBeginTime ( )

Returns the time when the timer was started.
### Return value

Begin time.
