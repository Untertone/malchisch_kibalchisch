# Unigine.EventConnections Class (CS)


This class is a container for the *[EventConnection](../../../../api/library/common/events/class.eventconnection_cs.md)* class used to store several *EventConnection* instances. The connections stored in one and the same *EventConnections* container may refer to different events.
