# Unigine::ImportLight Class (CS)


This class is an intermediate representation of a light source from a source file. The associated [ImportNode](../../../../api/library/common/import/class.importnode_cs.md) defines its position and orientation in the scene. During import, it is converted to a UNIGINE [Light](../../../../api/library/lights/class.light_cs.md) by an [import processor](../../../../api/library/common/import/class.importprocessor_cs.md).
