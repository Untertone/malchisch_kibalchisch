# Unigine::ImportProcessor Class (CS)


This class is used to manage import processors. An import processor is a module that saves objects generated by an [importer](../../../../api/library/common/import/class.importer_cs.md) on the basis of [metadata](../../../../api/library/common/import/class.importscene_cs.md) on the whole scene or some of its components to files in UNIGINE native format and performs auxiliary operations during the import process (data preparation, file management, etc.). You can use a set of different processors for each scene component or a single processor for all of them. There are also two specific types of processors:

- **Pre-processors** - perform additional operations with [scene metadata](../../../../api/library/common/import/class.importscene_cs.md) prior to generation of UNIGINE's objects.
- **Post-processors** - perform additional operations with files generated in the process of import (e.g. copying files to other folders, adding files to packages, etc.).


> **Notice:** This is a base class for all import processors. Your custom import processor class must be inherited from it.


You can override one (or all) *[onProcess*()](#onProcessAnimation_MeshSkinnedAnimation_ImportMeshSkinned_ImportAnimation_bool)* methods to define the types of components to be processed by it, these overridden methods will be called by the importer when corresponding import operations are performed.


> **Notice:** For pre- and post-processors only the *[onProcessScene()](#onProcessScene_ImportScene_bool)* method is to be overridden.

### Properties

The [importer](../../../../api/library/common/import/class.importer_cs.md) to be used for the import processor.

### Members

---

## ImportProcessor GetImportProcessor ( )

Returns the [import processor](../../../../api/library/common/import/class.importprocessor_cs.md) itself.
### Return value

Import processor itself.
## bool ProcessScene ( ImportScene scene )

Performs scene processing: modifies the metadata of [scene elements](../../../../api/library/common/import/class.importscene_cs.md) or files generated in the process of importing the scene.
> **Notice:** This method is used by [pre-processors and post-processors](#pre_post). To customize actions to be performed on scene processing, when implementing a custom pre- or post-processor, you can override the **[OnProcessScene()](../../../...md#onProcessScene_ImportScene_bool)** method.


### Arguments

- *[ImportScene](../../../../api/library/common/import/class.importscene_cs.md)* **scene** - [Scene](../../../../api/library/common/import/class.importscene_cs.md) to be processed.

### Return value

true if the specified scene is successfully processed; otherwise, false.
## bool ProcessTexture ( ImportTexture import_texture )

Performs texture processing: saves the specified generated texture to a corresponding file in the [output directory](#getOutputPath_cstr).
> **Notice:** To customize actions to be performed on texture processing, when [implementing a custom import processor](../../../../code/usage/custom_import_processor/index.md), you can override the **[OnProcessTexture()](../../../...md#onProcessTexture_ImportTexture_bool)** method.


### Arguments

- *[ImportTexture](../../../../api/library/common/import/class.importtexture_cs.md)* **import_texture** - Instance of the *[ImportTexture](../../../../api/library/common/import/class.importtexture_cs.md)* class containing information about the imported texture.

### Return value

true if the specified texture is successfully imported to a file; otherwise, false.
## bool ProcessMesh ( Mesh mesh , ImportMesh import_mesh )

Performs mesh processing: saves the specified generated mesh to a corresponding `*.mesh` file in the [output directory](#getOutputPath_cstr).
> **Notice:** To customize actions to be performed on mesh processing, when [implementing a custom import processor](../../../../code/usage/custom_import_processor/index.md), you can override the **[OnProcessMesh()](../../../...md#onProcessMesh_Mesh_ImportMesh_bool)** method.


### Arguments

- *[Mesh](../../../../api/library/rendering/class.mesh_cs.md)* **mesh** - [UNIGINE's mesh instance](../../../../api/library/rendering/class.mesh_cs.md) generated for the specified imported mesh.
- *[ImportMesh](../../../../api/library/common/import/class.importmesh_cs.md)* **import_mesh** - Instance of the *[ImportMesh](../../../../api/library/common/import/class.importmesh_cs.md)* class containing information about the imported mesh.
