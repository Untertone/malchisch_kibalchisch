# Unigine::ImportCamera Class (CS)


This class is an intermediate representation of a camera from a source file. The associated [ImportNode](../../../../api/library/common/import/class.importnode_cs.md) defines its position and orientation in the scene. During import, it is converted to a UNIGINE [Player](../../../../api/library/players/class.player_cs.md) by an [import processor](../../../../api/library/common/import/class.importprocessor_cs.md).
