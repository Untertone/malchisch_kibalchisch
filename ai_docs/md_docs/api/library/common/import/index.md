# File Import Functionality Classes


This section contains a set of basic classes implementing file import functionality. The whole [Import System](../../../../principles/import_system/index_cpp.md) is controlled via the import manager class - [Import Class](../../../../api/library/common/import/class.import_cpp.md).


### See Also


- The [Import System article](../../../../principles/import_system/index_cpp.md) for more details on the Import System
- The [Custom Import Plugin](../../../../code/usage/custom_import_plugin/index.md) usage example demonstrating how to use file import classes
- The [Importing Models Directly to Memory](../../../../code/usage/custom_import_processor/index.md) usage example to learn the basics of writing your own custom import processor.
