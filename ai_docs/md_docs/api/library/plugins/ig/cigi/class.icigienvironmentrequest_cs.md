# CigiEnvironmentRequest Class (CS)

**Inherits from:** CigiHostPacket


## CigiEnvironmentRequest Class

### Properties

## 🔒︎ int RequestID

The value of the **Request ID** parameter specified in the packet. Identifies the environmental conditions request.
## 🔒︎ int RequestType

The value of the **Request Type** parameter specified in the packet. Determines with which packet the IG shall respond.
## 🔒︎ dvec3 Position

The three-component vector of the values of the **Latitude, Longitude and Altitude** parameters specified in the packet. Specifies the geodetic latitude, longitude, and altitude at which the environmental state is requested.
