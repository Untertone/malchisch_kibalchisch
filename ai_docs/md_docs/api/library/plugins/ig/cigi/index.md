# CIGIConnector Plugin API

> **Warning:** The functionality described in this article is not available in the Community SDK edition.
> You should upgrade to [**Sim**](https://l.unigine.com/SdhugY462) SDK edition to use it.


This section contains a set of the CIGIConnector Plugin API Classes. The CIGIConnector Plugin implements the connection between the IG Plugin and the CIGI protocol.


For debugging IG-based projects using the CIGI protocol, [IG Host](../../../../../ig/ig_host.md) can be used.
