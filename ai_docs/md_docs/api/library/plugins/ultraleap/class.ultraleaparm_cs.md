# Unigine::Plugins::UltraleapArm Class (CS)


This structure represents a [forearm](../../../../code/plugins/ultraleap/index_cs.md#arms).


> **Notice:** [Ultraleap](../../../../code/plugins/ultraleap/index_cs.md) plugin must be loaded.


## UltraleapArm Class

### Properties

## 🔒︎ double Width

The width of the forearm, in meters.
## 🔒︎ vec3 Direction

The normalized direction in which the arm is pointing (from elbow to wrist).
## 🔒︎ vec3 PositionElbow

The coordinates of the elbow position.
## 🔒︎ vec3 PositionWrist

The coordinates of the wrist position.
## 🔒︎ vec3 Center

The coordinates of the center of the forearm.
## 🔒︎ UltraleapHand Hand

The object for the hand.
