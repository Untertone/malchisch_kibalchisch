# Unigine::Plugins::DataBridge::DataBridgeUI Class (CPP)

**Header:** #include <plugins/Unigine/DataBridge/UnigineDataBridge.h>


This class controls rendering of the list of all variables available in your application along with their types and current values.


![](../../../../code/plugins/databridge/databridgeui.png)


## DataBridgeUI Class

### Members

---

## void setVisible ( bool v )

Sets the value indicating if the DataBridge UI is rendered into the viewport.
### Arguments

- *bool* **v** - true to enable rendering of DataBridge UI, false to disable it.

## bool isVisible ( )

Returns the value indicating if the DataBridge UI is rendered into the viewport.
### Return value

true if DataBridge UI is visible, otherwise false.
