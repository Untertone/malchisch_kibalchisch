# Operating System Issues


This section contains recommendations on how to configure your operating system settings to avoid engine performance drops.


## Performance Drop


### Energy Efficiency Setting


If you develop an UNIGINE project in Linux, check the energy efficiency settings. It is noticed that an energy saving mode **Balanced** is set by default in the **Ubuntu 24.04.1 LTS** distribution. Due to this, the engine performance is reduced and developers are faced with spikes.


![](os_error_1.png)


To prevent spikes and improve the engine performance we recommend switching the settings to **Performance** mode.
